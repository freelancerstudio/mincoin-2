# jaeycoin

### Installation
* from scratch
```bash
~$ yarn init
# typescript
~$ yarn global add typescript
# support watch mode
~$ yarn add tsc-watch --dev
# crypto library
~$ yarn add crypto-j
# websocket library
~$ yarn add ws
```
* from cloning
```bash
~$ yarn install
```

### Run app
```bash
~$ yarn start
```
